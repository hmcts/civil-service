IvyArtifactSet

interface IvyArtifactSet : DomainObjectSet<T> (source)

A Collection of IvyArtifacts to be included in an IvyPublication. Being a DomainObjectSet, an IvyArtifactSet provides convenient methods for querying, filtering, and applying actions to the set of IvyArtifacts. 

plugins {
    id 'ivy-publish'
}

def publication = publishing.publications.create("my-pub", IvyPublication)
def artifacts = publication.artifacts

artifacts.matching({
    it.type == "source"
}).all({
    it.extension = "src.jar"
})
